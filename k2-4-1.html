LûËt oss ûËtervûÊnda till regressionsmodellen i ekvation 1. Vi berûÊknar fûÑrstagradsvillkoren genom att derivera uttrycket i ekvation 4 med hûÊnsyn till \(a\), \(b\) samt \(c\) var fûÑr sig och sûÊtta respektive resultat lika med 0. Eftersom vi har tre faktorer fûËr vi fûÑljande tre resultat: \(\frac{\partial}{\partial\widehat{a}}\left( \sum_{i = 1}^{n}\widehat{v_{i}^{2}} \right) = \sum_{}^{}{- 2\left( Y_{i} - \widehat{a} - \widehat{b}X_{i} - \widehat{c}Z_{i} \right)}\) (5)

\[\frac{\partial}{\partial\widehat{b}}\left( \sum_{i = 1}^{n}\widehat{v_{i}^{2}} \right) = \sum_{}^{}{- 2X_{i}\left( Y_{i} - \widehat{a} - \widehat{b}X_{i} - \widehat{c}Z_{i} \right)}\]

\[\frac{\partial}{\partial\widehat{c}}\left( \sum_{i = 1}^{n}\widehat{v_{i}^{2}} \right) = \sum_{}^{}{- 2Z_{i}\left( Y_{i} - \widehat{a} - \widehat{b}X_{i} - \widehat{c}Z_{i} \right)}\]

PûË liknande sûÊtt som vi gjorde i avsnitt 3.4 kan vi sûÊtta fûÑrsta villkoret lika med 0 och lûÑsa ut en definition fûÑr \(\widehat{a}\): \(0 = - 2\sum_{}^{}Y + \sum_{}^{}ûÂ + \sum_{}^{}\widehat{b}X_{i} + \sum_{}^{}{\widehat{c}Z_{i}}\) (6)

\[n\widehat{a} = nY - \widehat{b}nX_{i} - \widehat{c}nZ_{i}\]

\[\widehat{a} = \overline{Y} - \widehat{b}\overline{X} - \widehat{c}\overline{Z}\]

HûÊrifrûËn kan vi fortsûÊtta lûÑsa ut lutningskoefficienterna \(\widehat{b}\) och \(\widehat{c}\). HûÊr nûÑjer vi oss dock med att enbart jûÊmfûÑra slutresultaten, alltsûË deras estimatorer, och hoppar ûÑver sjûÊlva hûÊrledningarna.
